# Simon Game (Web Development Practice)

This is my Simon Game project built while learning web development.

## About the Project
This is a simple memory game inspired by the classic Simon game.

- The game shows a color pattern.
- The player must click the buttons in the same order.
- Each level adds one more color to the sequence.
- The game ends when a wrong button is clicked.

## Tech Used
- HTML
- CSS
- JavaScript (Vanilla JS)

## Project Structure
- `index.html` - Contains HTML, CSS, and JavaScript for the game.

## How to Run Locally
1. Download or clone this repository.
2. Open `index.html` in your browser.
3. Click **Start** to play.

## Learning Goal
I made this project to practice:

- Game state logic
- Basic UI design with CSS
